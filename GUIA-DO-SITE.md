## 🌐 1. Visão Geral do Site

O website da **Rui Figueira & Roque, Lda** é uma plataforma profissional que integra três áreas de negócio num único local:

```
┌─────────────────────────────────────────────────┐
│              RUI FIGUEIRA & ROQUE               │
│     Soluções integradas em Imobiliária,         │
│           Seguros e Crédito                     │
├─────────┬─────────────────┬─────────────────────┤
│  🏠     │   🛡️            │   🏦                │
│ Imóveis │   Seguros       │   Crédito Bancário  │
├─────────┴─────────────────┴─────────────────────┤
│              📧 Contacto                        │
│              👥 Sobre Nós                       │
└─────────────────────────────────────────────────┘
```

### Navegação Principal

O menu de topo está sempre visível e dá acesso a todas as secções:

```
╔══════════════════════════════════════════════════════════════════╗
║  Rui Figueira & Roque Lda    Início │ Imóveis │ Seguros │      ║
║                               Crédito │ Contatos  [Solicitar]  ║
╚══════════════════════════════════════════════════════════════════╝
```

| Botão              | O que faz                                        |
|---------------------|--------------------------------------------------|
| **Início**          | Volta à página principal                        |
| **Imóveis**         | Lista todos os imóveis disponíveis              |
| **Seguros**         | Mostra os tipos de seguros oferecidos           |
| **Crédito**         | Informações sobre crédito habitação e pessoal   |
| **Contatos**        | Formulário de contacto + informações            |
| **Solicitar Contacto** | Vai diretamente para o formulário de contacto |

> 📱 **No telemóvel**, o menu é substituído pelo ícone ☰ (hambúrguer) no canto superior direito.

## ⚙️ 8. Painel de Administração (Backoffice)

O backoffice é o sistema privado de gestão do site. Apenas o administrador tem acesso.

### 8.1 — Como Aceder

1. Aceda à página de **Login** (`/login`)
2. Introduza o **email** e a **password**
3. Clique em **"Entrar no Backoffice"**

```
┌──────────────────────────────────────────────────┐
│              BEM-VINDO DE VOLTA                  │
│                                                   │
│  Email:    [nome@exemplo.com          ]          │
│  Password: [••••••••••                ]          │
│                                                   │
│  ┌──────────────────────────────────┐            │
│  │        Entrar no Site            │  ← Abre    │
│  └──────────────────────────────────┘    o site  │

│  └──────────────────────────────────┘    admin   │
└──────────────────────────────────────────────────┘
```

> ⏰ A sessão expira automaticamente após **2 horas** de inatividade.

## ❓ 9. Perguntas Frequentes

### Para Visitantes

**Como posso agendar uma visita a um imóvel?**
> Na página do imóvel, preencha o formulário de contacto e selecione "Pedir Visita". A empresa será notificada imediatamente.

**Como peço uma simulação de seguro?**
> Vá à página **Seguros**, escolha o tipo pretendido e clique em "Fazer Simulação". Preencha os seus dados e será contactado.

**As minhas informações são seguras?**
> Sim. Todos os dados são transmitidos de forma encriptada e armazenados de forma segura. Consulte a nossa Política de Privacidade para mais detalhes.

### Para o Administrador

**Como adiciono um novo imóvel?**
> No backoffice, vá a **Imóveis** → **Novo Imóvel**. Preencha todos os campos e carregue as fotografias.

**Como coloco um imóvel em destaque na homepage?**
> Ao editar um imóvel, ative a opção **"Imóvel em Destaque"**. Apenas os 3 mais recentes com destaque aparecem na homepage.

**Como edito os textos do site?**
> No backoffice, vá a **Conteúdo**. Pode alterar todos os títulos, textos, contactos e fotos do site público.

**Porque não vejo as minhas alterações no site?**
> As páginas podem demorar alguns segundos a atualizar. Tente atualizar a página (F5 ou puxar para baixo no telemóvel).

**Como vejo os pedidos dos clientes?**
> No backoffice, vá a **Leads**. Todos os pedidos aparecem por ordem cronológica e pode filtrar por tipo e estado.
